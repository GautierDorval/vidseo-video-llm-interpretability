Features

VidSEO is designed to expose video meaning as readable text on the page.

Core publishing features

YouTube and Vimeo embeds

Use VidSEO to publish videos from major platforms with a shortcode-based workflow.

YouTube subtitles when available

VidSEO can reuse existing YouTube subtitles where they exist, instead of forcing a separate transcript process every time.

Manual transcripts

If subtitles are not available, you can add transcript content manually.

Native HTML transcript output

The transcript is rendered in the page as standard HTML that machines can parse directly.

Structured transcript content

Transcripts can be organized with headings, paragraphs, and links for clarity.

Editorial control

You remain in control of the transcript text and how it appears on the page.

Presentation options

  • show the transcript visibly on the page
  • keep the transcript in source while hiding it visually
  • place the transcript alongside the embedded video
  • insert the output wherever a shortcode can be used

Product boundary

These features make transcript content more available as text.

They do not mean that VidSEO:

  • interprets the transcript,
  • validates the transcript,
  • guarantees that a crawler will rank or cite the page,
  • replaces editorial judgment.
