# Features

VidSEO is designed to expose video meaning as readable text on the page.

## Core publishing features

<div class="card-grid">
  <div class="card">
    <h3>YouTube and Vimeo embeds</h3>
    <p>Use VidSEO to publish videos from major platforms with a shortcode-based workflow.</p>
  </div>
  <div class="card">
    <h3>YouTube subtitles when available</h3>
    <p>VidSEO can reuse existing YouTube subtitles where they exist, instead of forcing a separate transcript process every time.</p>
  </div>
  <div class="card">
    <h3>Manual transcripts</h3>
    <p>If subtitles are not available, you can add transcript content manually.</p>
  </div>
  <div class="card">
    <h3>Native HTML transcript output</h3>
    <p>The transcript is rendered in the page as standard HTML that machines can parse directly.</p>
  </div>
  <div class="card">
    <h3>Structured transcript content</h3>
    <p>Transcripts can be organized with headings, paragraphs, and links for clarity.</p>
  </div>
  <div class="card">
    <h3>Editorial control</h3>
    <p>You remain in control of the transcript text and how it appears on the page.</p>
  </div>
</div>

## Presentation options

- show the transcript visibly on the page
- keep the transcript in source while hiding it visually
- place the transcript alongside the embedded video
- insert the output wherever a shortcode can be used

## Product boundary

These features make transcript content more available as text.

They do **not** mean that VidSEO:
- interprets the transcript,
- validates the transcript,
- guarantees that a crawler will rank or cite the page,
- replaces editorial judgment.
