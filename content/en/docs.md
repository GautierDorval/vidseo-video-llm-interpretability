# Documentation

This page summarizes the public workflow and practical boundaries of VidSEO.

## Installation flow

1. Install VidSEO from WordPress.org or upload it manually.
2. Activate the plugin.
3. Open the VidSEO menu in WordPress admin.
4. Add a new video and its transcript.
5. Insert the shortcode into a post, page, or product.

## Publishing workflow

### Create a video entry

Start with the video URL and choose the platform.

### Add transcript content

Use one of two paths:
- retrieve existing YouTube subtitles when available,
- add transcript text manually.

### Publish the output

Insert the shortcode where you want the video and transcript to appear.

## Best-use guidance

- keep transcript text close to the video
- preserve clear paragraph structure
- avoid bloating the transcript with unrelated copy
- edit for accuracy before publication
- treat transcript text as content, not as decoration
