# How it works

VidSEO is built around a simple publishing workflow.

## 1. Choose the video source

Select the platform for the video you want to publish.

Supported references described publicly today include:
- YouTube
- Vimeo

## 2. Add the video URL

Paste the video URL into VidSEO.

## 3. Add or retrieve the transcript

You can:
- retrieve existing YouTube subtitles when available,
- or add the transcript manually.

## 4. Format the transcript

Use standard text structure to improve readability:
- headings
- paragraphs
- links
- section breaks

## 5. Insert the shortcode

Place the generated shortcode inside the post, page, or product where you want the video and transcript to appear.

## 6. Publish a page that is easier to read as text

The transcript is rendered as standard HTML in the page itself.
