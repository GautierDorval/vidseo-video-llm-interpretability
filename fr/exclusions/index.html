Exclusions

Exclusions explicites qui empêchent de lire VidSEO comme un système de génération IA ou de garantie de classement.

VidSEO possède un périmètre volontairement étroit.

VidSEO n’est pas

  • un moteur de transcription IA ;
  • un moteur de résumé ;
  • un moteur de réécriture ;
  • un moteur de vérification factuelle ;
  • une couche de certification d’autorité ;
  • un système de garantie de classement ;
  • un système de garantie de citation ;
  • un produit de gouvernance ou de monitoring des modèles ;
  • un remplacement du contrôle qualité éditorial.

Pourquoi des exclusions explicites comptent

Les dépôts dérivent lorsque les lecteurs comblent eux-mêmes les revendications absentes.

Dans le cas de VidSEO, les dérives risquées les plus fréquentes sont :

  • supposer que « LLM » signifie que VidSEO interprète les vidéos par IA ;
  • supposer que « SEO » signifie que VidSEO garantit des gains de classement ;
  • supposer que « lisible par machine » signifie que le contenu a été validé ;
  • supposer que la présence d’une transcription prouve expertise ou fiabilité.

Chacune de ces conclusions se situe hors du périmètre produit.

Frontière de revendication sûre

La revendication sûre est plus étroite :

VidSEO rend le texte de transcription vidéo disponible comme HTML natif dans la page afin que les machines puissent lire ce qui est explicitement dit.

Cette revendication est déjà assez forte.
Elle n’a pas besoin d’être gonflée.
