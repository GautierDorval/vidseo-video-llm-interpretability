Fonctionnement

VidSEO repose sur un parcours de publication simple.

1. Choisir la source vidéo

Sélectionner la plateforme de la vidéo à publier.

Les références publiques actuelles mentionnent :

  • YouTube
  • Vimeo

2. Ajouter l’URL de la vidéo

Coller l’URL dans VidSEO.

3. Ajouter ou récupérer la transcription

Deux possibilités :

  • récupérer des sous-titres YouTube existants quand ils sont disponibles,
  • ou ajouter une transcription manuellement.

4. Mettre en forme la transcription

Utiliser une structure textuelle standard pour améliorer la lisibilité :

  • titres
  • paragraphes
  • liens
  • sections

5. Insérer le shortcode

Placer le shortcode généré dans l’article, la page ou la fiche produit souhaitée.

6. Publier une page plus lisible comme texte

La transcription est rendue comme HTML standard dans la page elle-même.

Pas de couche d’inférence

VidSEO n’essaie pas d’interpréter le sens de la vidéo.

Pas de boîte noire

L’objectif est d’exposer le texte là où la page peut vraiment l’utiliser.

Pas de promesse gonflée

Publier du texte est concret. Le classement et la citation dépendent encore des systèmes externes.
