Fonctionnalités

VidSEO est conçu pour exposer le sens d’une vidéo comme texte lisible dans la page.

Fonctionnalités principales

Intégration YouTube et Vimeo

VidSEO permet de publier des vidéos de plateformes majeures via un flux fondé sur des shortcodes.

Sous-titres YouTube quand ils existent

VidSEO peut réutiliser des sous-titres YouTube existants au lieu d’imposer une transcription séparée à chaque fois.

Transcriptions manuelles

Si les sous-titres ne sont pas disponibles, il est possible d’ajouter le texte manuellement.

Sortie HTML native

La transcription est rendue dans la page comme HTML standard, lisible directement par les machines.

Structure du transcript

Le texte peut être organisé avec titres, paragraphes et liens pour améliorer la lisibilité.

Contrôle éditorial

Le publieur garde le contrôle sur le texte de transcription et sa présentation.

Options de présentation

  • afficher la transcription dans la page
  • garder la transcription dans le code source tout en la cachant visuellement
  • placer la transcription à côté de la vidéo
  • insérer la sortie partout où un shortcode peut être utilisé

Limite produit

Ces fonctionnalités rendent le contenu vidéo plus disponible comme texte.

Elles ne signifient pas que VidSEO :

  • interprète la transcription,
  • valide la transcription,
  • garantit qu’un robot classera ou citera la page,
  • remplace le jugement éditorial.
