# Governance

This file governs the **repository**, not the editorial subject of the public site.

## Core rule

`vidseo.dev` must remain a site about **VidSEO** before it is anything else.

## Editorial separation

### Primary subject of the public site

- VidSEO as a WordPress plugin
- product capabilities
- workflow
- documentation
- use cases
- FAQs
- technical boundaries

### Secondary / structural layer

- machine-readable metadata
- repository policy
- security and support
- contribution workflow
- CI validation

## Anti-drift rule

Contributors must not let the public home page drift into:
- governance-first framing,
- policy-first framing,
- machine-file-first framing.

Machine-readable surfaces may be exhaustive, but they are **supporting infrastructure**, not the site's editorial center.
